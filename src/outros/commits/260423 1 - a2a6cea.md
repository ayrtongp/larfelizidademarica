### `src/components/Residentes/Suprimentos.tsx`

- **Empty states** limpos; sem `notifyError` para listas vazias

## 6. Autenticação — renovação automática de token

### `src/pages/api/auth/refresh.ts` *(novo)*

- Endpoint `POST /api/auth/refresh` — recebe o token atual no header `Authorization: Bearer`, verifica com `jwt.verify` e devolve novo token com validade de 1h

### `src/components/CheckToken.tsx`

- Adicionado `tryRefreshToken`: quando o token expira em menos de 20 minutos (`REFRESH_THRESHOLD_MIN = 20`), chama `/api/auth/refresh` automaticamente e substitui o token no `localStorage` sem deslogar o usuário
