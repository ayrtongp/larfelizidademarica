## 7. Fix: Checkbox de lista de compras no mobile

### Problema
Cada checkbox marcado causava "refresh" visual por dois motivos:
1. `toggleComprado` bloqueava com `await onSave()` + spinner + disabled em todos checkboxes
2. `handleSalvarItens` chamava `carregarLista()` no erro, que fazia `setLoading(true)` desmontando a página

### Correção
- `toggleComprado` agora é fire-and-forget (não bloqueia)
- `handleSalvarItens` faz revert silencioso em vez de reload
- `toggleItemComprado` — novo endpoint atômico que altera só `itens.$.comprado` via operador posicional do MongoDB (seguro para uso simultâneo por duas pessoas)
